## Sobre o Projeto

Este repositório reúne exercícios desenvolvidos durante o curso de JavaScript realizado na faculdade, como parte de atividades avaliativas e práticas da disciplina.

Os exercícios têm como objetivo reforçar os conceitos fundamentais da linguagem JavaScript, incluindo:

Lógica de programação

Manipulação de dados

Interação entre HTML e JavaScript

Resolução de problemas práticos

O curso foi concluído com certificado, e este repositório serve como registro do aprendizado e evolução acadêmica, além de apoio para revisões futuras e demonstração de conhecimentos básicos em JavaScript.
